# marketplace

## Project setup
In root folder, /marketplace, do

```
npm install
```

In book-server folder, /marketplace/book-server, do

```
npm install
```

### Compiles and hot-reloads for development

using Concurrently to run both frontend and backend.
```
npm run start
```
